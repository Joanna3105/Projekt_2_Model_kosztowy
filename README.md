# Projekt 2: Analiza kosztów i budżetu w Microsoft Excel

## Opis projektu

Projekt dotyczy analizy kosztów w układzie:
- wykonanie kosztów (2025–2026),
- budżet 2026,
- odchylenia budżetowe,
- struktura kosztów wg grup kosztowych,
- analiza centrów kosztowych (MPK),
- analiza trendu oraz wykonania w ujęciu YTD.

## Zakres modelu danych

- budżet,
- koszty,
- plan kont (MPK, grupy kosztowe, dysponent),
- tabela kalendarz (model czasu).

## Zastosowane narzędzia analityczne

Projekt dotyczy analizy kosztów w układzie:
- wykonanie kosztów (2025–2026),
- budżet 2026,
- odchylenia budżetowe,
- struktura kosztów wg grup kosztowych,
- analiza centrów kosztowych (MPK),
- analiza trendu oraz wykonania w ujęciu YTD.

## Zakres prac

- przygotowanie danych w Power Query i stworzenie Kalendarza,
- utworzenie relacji i miary DAX w Power Pivot,
- budowa dashboardu: tabele przestawne i wykresy,
- fragmentatory,
- wnioski i rekomendacje na aktualny zestaw danych.
